Concevez votre application mobile avec Synthetic !

En quelques lignes...

Facile à apprendre

Synthetic est un langage de développement orienté mobile design. Avec le concept des mixin, son objectif est plutot de construire des interfaces finalisées sur du HTML, en parsant en second lieu une forme logique de code HTML [LogicHTML (*.lh)]

En savoir plus

Facile à installer

$ npm install -g synthetic-dev

Facile à mettre en place

$ synthetic init [DIRECTORY]

Une architecture simple


    @import * from sspa;

    @use sspa.Layer(
        $appbar: @use sspa.AppBar(
            $title: About,
            $has_back: true
        ),
        $view : @use sspa.Center(
            $children : [
                @use sspa.Text('Hello world !', 
                    $align: center
                )
            ]
        ),
        $floatButtons : @use sspa.FloatWrapper(
            $children : [
                @use sspa.FloatButton(
                    $icon : @use sspa.Icon('plus'),
                    $background: $super.theme,
                    $name : 'add',
                    $event : {
                        tap : (){
                            console.log('[tapped]');
                        }
                    }
                )
            ]
        ),
    );
